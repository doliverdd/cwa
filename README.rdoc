= cwa_as Cluster Web Access Account Signup

This plugin will provide a means for new users, after Single-Sign-On auth, to

1. Agree to terms-of-service (configurable via wiki)

2. Agree to system access agreement (configurable via wiki)

3. Grab Single-Sign-On password to generate JSON-RPC request to create a new IPA user

4. Provide for account "deletion" with terms-of-discontinuation-of-service (wiki)
